Comandos en GIT

Git es un Software de control de versiones pensado en la eficiencia, confiabilidad y compatibilidad del mantenimiento de versiones de aplicaciones cuando estas tienen un gran número de archivos de código fuente

git init: inicializa un nuevo repositorio Git

Añade un directorio de trabajo en el entorno de ensayo

git commit: Crea una versión en el repositorio local, incluyendo los cambios procedentes del area de trabajo
